# Phasmophobia web APP (unoffical)

This little application helps you to identify the correct ghost by combining or exclude somes evidences.

Like the in-game manual, you can filter remaining ghots possibilities by check to "YES" evidences found.
But also have a view of wich combinations of evidences are possible or not. 
You can also check evidences as "NO"  to exclude ghost with this one.

This APP also provide a timer to keep on eye on it during the ghost research.

Avialable in French / English / Partially Japanese 

/!\ This is not a cheat ! Just an interactive guide !

## Example 01 : Combining
![Img 01](https://phasmophobia.bidoum.fr/img/Screenshot01.jpg)

You found two evidences:
- Ghost writing
- Freezing temperature

This app display that there is only two possiblities : Demon or Yurei
It also tells you that you don't need to search "EMF" or "Fingerprint" but "Spirit box" or "Ghost orb"

## Example 02 : Exclude evidences
![Img 01](https://phasmophobia.bidoum.fr/img/Screenshot02.jpg)

You found one evidence : "Fingerprints"

You're sure there is no "Spirit box" after having tried to call him during 5mn...

The APP filter the ghost with "Fingerprint" and no "Spirit box" to found two results : "Banshee" and "Revenant", and then is able to directly tell you to search for "Book" "Temperature" or "EMF" but not for "Ghost Orb"
